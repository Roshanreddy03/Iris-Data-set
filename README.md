# Iris-Data-cleaning

Project Overview:

- Performed an Exploratory Data Analysis (EDA) on the classic Iris dataset.

- The primary goal was to understand and visualize how the three flower species (setosa, versicolor, virginica) differ based on their physical measurements.

- Using Python and data analysis techniques, we perform exploratory data analysis (EDA) to uncover patterns and insights from the dataset.

Link to Project Page: https://roadmap.sh/projects/exploring-iris-dataset

Dataset

Source: Iris Dataset

File used: iris.csv

-------------------------------------------------------------------------------------------------------------------------------------------------------------

Technologies Used:

- Python

- Pandas

- Seaborn 

- Matplotlib

- Jupyter Notebook

-------------------------------------------------------------------------------------------------------------------------------------------------------------

Data Exploration:

- Verified that the dataset contains an equal number of samples (50) for each of the three species.

- Generated histograms and box plots to observe the spread and identify potential outliers for all numerical features.

- Created a pairplot and a correlation heatmap to analyze how the features interact with one another across different species.

-------------------------------------------------------------------------------------------------------------------------------------------------------------

Key Insights:

- Iris setosa is unique: It forms a completely separate cluster from the other two species, characterized by much smaller petals.

- Some overlap exists: Iris versicolor and Iris virginica share similar sepal sizes and have slight overlap in petal sizes, though virginica is generally larger.

- Strong correlation: Petal length and petal width are highly positively correlated, meaning as one gets larger, the other does too.

-------------------------------------------------------------------------------------------------------------------------------------------------------------

Visualisations:

- Create Histograms and Box Plots

- Visualize Relationships with a Pairplot

- Compute Correlation Matrix and Heatmap

-------------------------------------------------------------------------------------------------------------------------------------------------------------

Conclusion:

The analysis successfully shows that petal measurements (length and width) are the most reliable features for distinguishing between the three Iris species, making them excellent predictors for any future machine learning models.
